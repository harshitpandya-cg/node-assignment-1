# Student CGPA API

## Objective
Build a REST API using Express.js to manage student CGPA records stored in-memory.

## Routes Implemented

GET /students  
GET /students/topper  
GET /students/average  
GET /students/count  
GET /students/:id  
GET /students/branch/:branchName  

## Sample URL
https://node-assignment-1-f80h.onrender.com/students

## Run Locally
npm install  
npm start
